Contenido

Esta página web está diseñada para facilitar la gestión eficiente de cursos académicos.

La página cuenta con una barra de navegación intuitiva que permite acceder fácilmente a diferentes secciones principales: Inicio, Cursos, Agregar Curso y Contacto.

La sección principal destaca la gestión de cursos, mostrando una lista ordenada de cursos activos.

La página incorpora un formulario de registro de cursos, donde usuarios pueden ingresar nuevos cursos con detalle. Este formulario cuenta con validaciones en tiempo real para que los cursos sean ingresados correctamente.

La página ha sido realizada por Diego Felipe Peralta Peralta, estudiante de quinto ciclo de la carrera de computación en la materia de Programación y plataformas web. Se puede encontrar los enlaces de comunicación en la sección de contacto.
